Section 3: Practical Project

Portfolio Website Creation - 10 Marks

Create a Personal Portfolio Website

Project Requirements (10 Marks Total)

  • HTML Structure (3 marks): Proper HTML5 semantic structure with header, main, sections, and footer
  • CSS Styling (3 marks): Professional styling with colors, fonts, layout, and responsive design
  • Content Sections (2 marks): Include About, Skills, Projects, and Contact sections
  • Navigation (1 mark): Working navigation menu that links to different sections
  • Code Quality (1 mark): Clean, well-organized, and commented code

Technical Specifications

  • Use HTML5 semantic elements (header, nav, main, section, footer)
  • Implement CSS Flexbox or Grid for layout
  • Include at least one CSS animation or transition
  • Make the website responsive (mobile-friendly)
  • Use external CSS file (not inline styles)
  • Include proper meta tags for SEO

Content Guidelines

  • Header: Your name and professional tagline
  • About Section: Brief introduction about yourself
  • Skills Section: Display your web development skills
  • Projects Section: Showcase 2-3 sample projects
  • Contact Section: Your contact information and social links
  • Footer: Copyright information
